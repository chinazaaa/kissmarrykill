# Architecture & SOLID debt tracker

Living checklist from the 2026-06 architecture audit (3 parallel reviews over `dev`). Tick items as they land. Severity = impact, Effort = S/M/L.

**Root cause:** there is no game registry ÔÇö each of ~29 games is wired by hand-maintained `if (isXGameÔÇŽ)` chains, so adding one game touches **~22 files / 40+ edit-sites**. Most items below are symptoms of that. Recommended order is top-to-bottom (bugs Ôćĺ tests Ôćĺ exhaustiveness Ôćĺ registry Ôćĺ decomposition); the later structural work is unsafe without the earlier safety nets.

Legend: `[ ]` todo ┬Ě `[x]` done ┬Ě `[~]` in progress
